# BrainNet-REALTIME-MRI-ANALYSIS-USING-DEEP-LEARNING-MODELS
BrainNet AI is a dual-model Deep Learning system that classifies brain MRI scans using EfficientNet and ResNet for improved diagnostic accuracy.It introduces novel Spatial Consensus Confidence (SCC), which uses between GradCAM++ activation maps to verify both models focus on the same brain region, making predictions more trustworthy.
